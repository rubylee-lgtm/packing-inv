Packing List / Invoice 產生工具

v1.1.0

僅能匯入 CSV 檔案（產品目錄 Catalog、訂單查詢結果），依照選擇的模式 Packing List 或 Invoice， 在前端直接產生 .xlsx 檔供下載。

不同模式會套用不同的明細欄位與計算邏輯。
僅接受 .csv。預期包含 Barcode、G.W. (KGS)、Product Name 等欄位； 第一列可為標題，程式會自動尋找真正的表頭。
需包含欄位： retailer_erp_customer_name、retailer_address、 retailer_contact_phone、retailer_contact_name、 sku、quantity、pcs_per_carton、 carton、effective_date、measurement、 Net Weight， 以及（Commercial Invoice 模式）quoted_price_currency。
4. 訂單內容預覽（最多顯示前 100 列）

尚未選擇檔案。

5. 轉換並下載 Excel
會先讀取 Catalog 建立字典（Barcode → { gw, name, unitPrice }），再解析訂單並依模式產出 Packing List 或 Commercial Invoice。
